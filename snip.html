Prototype Pollution

All versions of lodash versions before 4.17.21 are vulnerable to Prototype pollution via the toObject converter. This allows attackers to inject arbitrary properties on Object.prototype which may lead to Denial of Service or Remote Code Execution in specific circumstances.

CVE ID(s): CVE-2021-23337

CWE ID(s): CWE-1321

CVSS: CVSS:3.1/AV:N/AC:L/PR:N/UI:N/S:U/C:H/I:H/A:H

Suggested Fix: app@1.0.0 → lodash@4.17.21
